# 快捷项目日报

一款专为工程项目管理人员设计的移动端日报应用，帮助快速记录和汇总每日工作进展。

## 功能特性

### 📋 多项目管理
- 支持多标签页同时管理多个项目
- 项目名称自动保存历史记录，方便快速输入
- 每个项目数据独立存储，互不影响

### 📝 日报内容
- **基本信息**：项目名称、日期、天气
- **人员统计**：管理人员数量、工人人数（便捷计数器）
- **进度追踪**：拖拽式进度条，实时显示进度百分比
- **今日完成**：记录当日已完成的工作事项
- **明日计划**：规划次日工作计划
- **问题需求**：记录遇到的问题和待解决需求

### ⚡ 便捷功能
- **一键生成**：点击按钮自动生成完整日报并复制到剪贴板
- **自动保存**：输入内容自动保存为草稿，断电不丢失
- **跨天智能结转**：自动将"明日计划"转为次日"今日完成"
- **多项目汇总**：一键汇总所有项目的日报内容
- **复制单条记录**：快速复制任意字段内容

### 📱 移动端体验
- 专为手机设计的简洁界面
- 触觉震动反馈，操作更有手感
- 响应式布局，适配各种屏幕尺寸

## 技术栈

- **前端**：原生 HTML + CSS + JavaScript
- **移动框架**：Capacitor 6.0
- **目标平台**：Android

## 项目结构

```
DailyReportApp/
├── app.js                 # 应用核心逻辑
├── index.html            # 主页面结构
```

## 构建说明

### 环境要求
- Node.js 16+
- Android SDK

### 安装依赖

```bash
npm install
```

### 添加 Android 平台

```bash
npx cap add android
```

### 开发运行

```bash
npx cap open android
```

### 构建 APK

```bash
npx cap sync
npx cap copy
# 在 Android Studio 中选择 Build > Build APK
```

## 自动构建

本项目配置了 GitHub Actions 自动化构建，每次推送代码到 `main` 分支时会自动构建 APK。

构建产物可在 Actions 页面下载。

## 使用场景

适用于建筑工地、工程项目、施工现场等需要每日汇报工作进展的场景。
